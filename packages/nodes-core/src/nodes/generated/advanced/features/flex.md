# Flex Node

**Category:** Advanced / Features

Flex solid for living hinges

### Bend Angle

- **Type:** number
- **Default:** 90
- **Min:** 0
- **Max:** 180

- **Type:** number
- **Default:** 10

- **Max:** 1000

- **Type:** number
- **Default:** 1
- **Min:** 0.1
- **Max:** 10

- **Required:** Yes

### Trim Planes
