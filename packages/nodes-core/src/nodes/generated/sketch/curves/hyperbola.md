# Hyperbola Node

**Category:** Sketch / Curves

Create a hyperbolic curve

- **Default:** 50

- **Max:** 10000

### Minor Radius

- **Type:** number
- **Default:** 30
- **Min:** 0.1
- **Max:** 10000

- **Default:** -2
- **Min:** -10
- **Max:** 10

### End Param

- **Type:** number
- **Default:** 2
- **Min:** -10
- **Max:** 10

- **Type:** Point
