# Parabola Node

**Category:** Sketch / Curves

Create a parabolic curve

## Parameters

### Focal Length

- **Type:** number
- **Default:** 10
- **Min:** 0.1
- **Max:** 10000

### Start Param

- **Type:** number
- **Default:** -100
- **Min:** -10000
- **Max:** 10000

### End Param

- **Type:** number
- **Default:** 100
- **Min:** -10000
- **Max:** 10000

## Inputs

### Vertex

- **Type:** Point
- **Required:** No

## Outputs

### Curve

- **Type:** Wire
- **Description:** Parabolic curve
