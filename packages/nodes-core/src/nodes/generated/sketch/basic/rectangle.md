# Rectangle Node

**Category:** Sketch / Basic

Create a rectangle

## Parameters

### Center X

- **Type:** number
- **Default:** 0
- **Min:** -10000
- **Max:** 10000

### Center Y

- **Type:** number
- **Default:** 0
- **Min:** -10000
- **Max:** 10000

### Width

- **Type:** number
- **Default:** 100
- **Min:** 0.1
- **Max:** 10000

### Height

- **Type:** number
- **Default:** 50
- **Min:** 0.1
- **Max:** 10000

### Filled

- **Type:** boolean
- **Default:** true

- **Description:** Create as face (filled) or wire (outline)

### Corner Radius

- **Type:** number
- **Default:** 0
- **Min:** 0
- **Max:** 1000
- **Description:** Corner rounding radius

## Inputs

This node has no inputs.

## Outputs

### Shape

- **Type:** Shape
- **Description:** Rectangle face or wire
