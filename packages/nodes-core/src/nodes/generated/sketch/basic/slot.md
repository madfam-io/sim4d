# Slot Node

**Category:** Sketch / Basic

Create a slot (rounded rectangle)

## Parameters

### Center X

- **Type:** number
- **Default:** 0
- **Min:** -10000
- **Max:** 10000

### Center Y

- **Type:** number
- **Default:** 0
- **Min:** -10000
- **Max:** 10000

### Length

- **Type:** number
- **Default:** 100
- **Min:** 0.1
- **Max:** 10000

### Width

- **Type:** number
- **Default:** 20
- **Min:** 0.1
- **Max:** 10000

### Angle

- **Type:** number
- **Default:** 0
- **Min:** -180
- **Max:** 180
- **Description:** Rotation angle

## Inputs

This node has no inputs.

## Outputs

### Face

- **Type:** Face
- **Description:** Slot face
