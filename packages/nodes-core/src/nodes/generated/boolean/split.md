# Split Node

**Category:** Boolean

Split shapes by each other

## Parameters

### Keep All

- **Type:** boolean
- **Default:** true

- **Description:** Keep all fragments

## Inputs

### Shapes

- **Type:** Shape[]
- **Required:** Yes
- **Description:** Shapes to split

### Tools

- **Type:** Shape[]
- **Required:** Yes
- **Description:** Splitting tools

## Outputs

### Fragments

- **Type:** Shape[]
- **Description:** Split fragments
