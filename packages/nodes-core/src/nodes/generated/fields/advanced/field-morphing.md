# FieldMorphing Node

**Category:** Fields / Advanced

Morph between two fields

## Parameters

### Factor

- **Type:** number
- **Default:** 0.5
- **Min:** 0
- **Max:** 1
- **Description:** Morphing factor (0=field1, 1=field2)

### Interpolation

- **Type:** enum
- **Default:** "\"linear\""

- **Description:** Interpolation method

## Inputs

### Field1

- **Type:** Field
- **Required:** No

### Field2

- **Type:** Field
- **Required:** No

## Outputs

### Morphed Field

- **Type:** Field
