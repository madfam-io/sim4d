# SphericalBearing Node

**Category:** MechanicalEngineering / Bearings

Create spherical bearing for misalignment

## Parameters

### Ball Diameter

- **Type:** number
- **Default:** 20
- **Min:** 5
- **Max:** 100

### Bore Diameter

- **Type:** number
- **Default:** 8
- **Min:** 3
- **Max:** 50

### Housing Diameter

- **Type:** number
- **Default:** 30
- **Min:** 10
- **Max:** 150

### Misalignment Angle

- **Type:** number
- **Default:** 15
- **Min:** 5
- **Max:** 30

## Inputs

### Center

- **Type:** Point
- **Required:** Yes

## Outputs

### Bearing

- **Type:** Shape

### Ball

- **Type:** Shape

### Housing

- **Type:** Shape
