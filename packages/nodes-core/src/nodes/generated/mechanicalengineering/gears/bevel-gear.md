# BevelGear Node

**Category:** MechanicalEngineering / Gears

Create bevel gear for angle transmission

- **Type:** number
- **Default:** 3
- **Min:** 1
- **Max:** 20

### Teeth

- **Type:** number
- **Default:** 25
- **Min:** 10
- **Max:** 100

### Cone Angle

- **Type:** number
- **Default:** 45
- **Min:** 10
- **Max:** 80

- **Type:** number
- **Default:** 15
- **Min:** 5
- **Max:** 50
