# QuadMesh Node

**Category:** Mesh / Tessellation

Generate quad-dominant mesh

## Parameters

### Target Quad Size

- **Type:** number
- **Default:** 5
- **Min:** 0.1
- **Max:** 100

### Quad Dominance

- **Type:** number
- **Default:** 0.8
- **Min:** 0
- **Max:** 1

## Inputs

### Shape

- **Type:** Shape
- **Required:** Yes

## Outputs

### Quad Mesh

- **Type:** Mesh
