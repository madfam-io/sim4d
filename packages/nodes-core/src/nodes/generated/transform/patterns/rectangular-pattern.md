# RectangularPattern Node

**Category:** Transform / Patterns

Creates a 2D rectangular grid pattern

## Parameters

### Count X

- **Type:** number
- **Default:** 4
- **Min:** 1
- **Max:** 100

### Count Y

- **Type:** number
- **Default:** 3
- **Min:** 1
- **Max:** 100

### Spacing X

- **Type:** number
- **Default:** 20
- **Min:** 0.1
- **Max:** 10000

### Spacing Y

- **Type:** number
- **Default:** 20
- **Min:** 0.1
- **Max:** 10000

### Staggered

- **Type:** boolean
- **Default:** false

- **Description:** Stagger alternate rows

## Inputs

### Shape

- **Type:** Shape
- **Required:** Yes

## Outputs

### Shapes

- **Type:** Shape[]

### Compound

- **Type:** Shape
