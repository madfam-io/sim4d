# Rotate Node

**Category:** Transform

Rotate shape around axis

## Parameters

### Angle

- **Type:** number
- **Default:** 45
- **Min:** -360
- **Max:** 360
- **Description:** Rotation angle in degrees

### Axis X

- **Type:** number
- **Default:** 0
- **Min:** -1
- **Max:** 1
- **Description:** Axis X component

### Axis Y

- **Type:** number
- **Default:** 0
- **Min:** -1
- **Max:** 1
- **Description:** Axis Y component

### Axis Z

- **Type:** number
- **Default:** 1
- **Min:** -1
- **Max:** 1
- **Description:** Axis Z component

### Center X

- **Type:** number
- **Default:** 0
- **Min:** -10000
- **Max:** 10000

### Center Y

- **Type:** number
- **Default:** 0
- **Min:** -10000
- **Max:** 10000

### Center Z

- **Type:** number
- **Default:** 0
- **Min:** -10000
- **Max:** 10000

### Copy

- **Type:** boolean
- **Default:** true

## Inputs

### Shape

- **Type:** Shape
- **Required:** Yes

## Outputs

### Rotated

- **Type:** Shape
- **Description:** Rotated shape
