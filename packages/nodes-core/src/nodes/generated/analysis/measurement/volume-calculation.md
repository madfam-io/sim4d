# VolumeCalculation Node

**Category:** Analysis / Measurement

Calculate volume and mass properties

## Parameters

### Precision

- **Type:** number
- **Default:** 0.01
- **Min:** 0.001
- **Max:** 1

### Density

- **Type:** number
- **Default:** 1
- **Min:** 0.001
- **Max:** 100
- **Description:** Material density

### Solid

- **Type:** Shape
- **Required:** Yes

### Volume

- **Type:** number

### Mass

- **Type:** number

### Center Of Mass

- **Type:** Point

### Inertia Matrix
