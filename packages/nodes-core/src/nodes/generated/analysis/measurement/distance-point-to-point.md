# DistancePointToPoint Node

**Category:** Analysis / Measurement

Measure distance between two points

## Parameters

This node has no parameters.

## Inputs

### point1

- **Type:** Point
- **Required:** Yes

### point2

- **Type:** Point
- **Required:** Yes

## Outputs

### distance

- **Type:** number
- **Description:** Distance value

### vector

- **Type:** Vector
- **Description:** Vector from point1 to point2
