# DistancePointToPlane Node

**Category:** Analysis / Measurement

Measure distance from point to plane

## Parameters

This node has no parameters.

## Inputs

### point

- **Type:** Point
- **Required:** Yes

### plane

- **Type:** Face
- **Required:** Yes

## Outputs

### distance

- **Type:** number
- **Description:** Perpendicular distance

### projectedPoint

- **Type:** Point
- **Description:** Projected point on plane
