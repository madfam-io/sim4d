# MinimumDistance Node

**Category:** Analysis / Proximity

Find minimum distance between geometries

## Parameters

### Tolerance

- **Type:** number
- **Default:** 0.01
- **Min:** 0.001
- **Max:** 1

### Show Connection

- **Type:** boolean
- **Default:** true

## Inputs

### Geometry1

- **Type:** Shape
- **Required:** Yes

### Geometry2

- **Type:** Shape
- **Required:** Yes

## Outputs

### Distance

- **Type:** number

### Point1

- **Type:** Point

### Point2

- **Type:** Point

### Connection Line

- **Type:** Wire
