# ProximityAnalysis Node

**Category:** Analysis / Proximity

Analyze proximity between multiple objects

## Parameters

### Threshold

- **Type:** number
- **Default:** 1
- **Min:** 0.1
- **Max:** 100

### Show Connections

- **Type:** boolean
- **Default:** true

## Inputs

### Objects

- **Type:** Shape[]
- **Required:** Yes

## Outputs

### Proximity Pairs

- **Type:** Shape[][]

### Distances

- **Type:** number[]

### Connections

- **Type:** Wire[]
