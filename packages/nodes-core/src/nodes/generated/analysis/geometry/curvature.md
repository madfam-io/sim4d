# Curvature Node

**Category:** Analysis / Geometry

Analyze curvature at a point

## Parameters

### parameter

- **Type:** number
- **Default:** 0.5
- **Min:** 0
- **Max:** 1

## Inputs

### curve

- **Type:** Wire
- **Required:** Yes

## Outputs

### curvature

- **Type:** number
- **Description:** Curvature value

### radius

- **Type:** number
- **Description:** Radius of curvature

### center

- **Type:** Point
- **Description:** Center of curvature

### normal

- **Type:** Vector
- **Description:** Normal vector
